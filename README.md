# PopX React App

This project is a simple mobile-style user onboarding application built using React and Vite. It allows users to create an account, log in, and view their account information in a clean and responsive interface.

## Features

* User registration form
* Login page
* Account settings page displaying user details
* Client-side routing using React Router
* Data stored using browser localStorage
* Mobile-centered UI layout

## Tech Stack

* React
* Vite
* React Router
* CSS

## Live Demo

https://popx-project-orpin.vercel.app/

## Installation

Clone the repository and run:

npm install
npm run dev

## Deployment

The project is deployed on Vercel for fast and reliable hosting.

## Project Purpose

This project demonstrates a simple authentication UI flow and routing implementation in a React application.
